# Sumsung-Supply-Chain-Logistic-Dashboard
This Power BI dashboard provides a clear view of Samsung’s supply chain performance across suppliers, inventory, shipments, and customers. It is designed to help stakeholders monitor key metrics, identify risks, and make informed operational decisions.

The report includes multiple pages (Home, Overview, Supplier, Inventory, Shipment, Customer) as shown in the uploaded file 
**supply chain & Logistic powerBI**

**Key Metrics:**
From the Overview page:
- Revenue: 173.26M
- Profit: 47.69M
- Profit Margin: 27.53%
- Perfect Order Rate: 75.29%
- Orders: 129K
- Inventory: 160K
- Shipments: 3M

**Dashboard Highlights:**

**Supplier**
- Samsung Vietnam leads by order volume
- Avg. lead time: 10–12 days
- Orders concentrated among key suppliers

**Inventory**
- Total stock (159K) above safety stock (88K) → low stock risk
- Defect rate peaks in March & October
- Top 3 products drive most defects

**Shipment**
- 75% on-time delivery
- Maersk has the highest delays
- Shipment costs peak mid-year

**Customer**
- Amazon is the top revenue contributor
- Online channel generates the highest profit share
- Profit peaks in Q4

**Data & Modeling:**
- Data from supplier, inventory, shipment, and sales systems
- Transformed using Power Query
- Star schema model with fact and dimension tables
- DAX measures created for KPIs and performance tracking

**How to Use:**

- Navigate using page buttons (Overview, Supplier, Inventory, etc.)
- Use slicers to filter by category or date
- Click visuals for cross-filtering
- Hover for tooltips and use drill-down where available
- Export data directly from visuals

**Setup**
- Open supply chain & Logistic.pbix in Power BI Desktop
- Configure data source credentials if required
- Publish to Power BI Service for scheduled refresh

This dashboard provides a consolidated view of supply chain performance, helping improve supplier efficiency, inventory planning, shipment reliability, and revenue optimization.
